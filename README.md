# Аналитика интернет-магазина
Проект по курсу аналитики данных. Система сбора и анализа данных для интернет-магазина.

## Структура проекта
- **PostgreSQL:** порт `5432`. База данных `analytics_db` со схемой магазина (заполняется генератором).
- **Генератор данных** (`generator/main.py`): создаёт заказы с привязкой к клиентам и пунктам выдачи. Работает автоматически.
- **Redash:** визуализация данных, доступен на порту `5000`.

## Запуск

**Одна команда** (из корня проекта):

```bash
docker compose up -d
```

При первом запуске будут собраны образы (Если вносишь изменения в генератор, пересоберите: `docker compose up -d --build`).

- **Redash:** http://localhost:5000 — при первом запуске создайте учётную запись администратора.
- **PostgreSQL (В Redash добавьте Data Source → PostgreSQL: Host `db`, Port `5432`, User/Password `user`/`pass` (по умолчанию), Database `analytics_db`.
